## Chatbot using Hugging Face Transformers

This project implements a simple conversational chatbot using a pre-trained transformer model (FLAN-T5).

### Features:
- Accepts user input from console
- Generates meaningful responses using NLP model
- Maintains interactive conversation
- Stops when user types 'exit' or 'quit'

### Technologies Used:
- Python
- Hugging Face Transformers
- PyTorch

### Model Used:
- google/flan-t5-base

### Working Pipeline:
User Input → Tokenization → Model Processing → Response Generation → Output Display
